# Ryfacto website

## Getting started

- Install Ruby 3.4.9 (using rbenv `brew install rbenv` then `rbenv install 3.4.9` then `rbenv global 3.4.9`)
- Install gems using `bundle install`.
- To run locally : `bundle exec jekyll serve`.
